# 🏥 Health Risk Prediction Using XGBoost

---

## 📌 Project Overview

This project develops a **Supervised Machine Learning Classification Model** to predict whether an individual is **Healthy** or **Unhealthy** based on physiological measurements, lifestyle habits, and medical history.

The solution simulates a real-world healthcare use case where biomedical researchers can identify high-risk individuals for clinical studies and population health analysis.

---

# 🎯 Business Problem

Biomedical research institutes collect thousands of health records every year.

The objective is to build a predictive model capable of classifying individuals into:

- ✅ Healthy
- ⚠️ Unhealthy

using multiple health-related indicators.

# 🤖 Model Used

### 🚀 XGBoost Classifier

The project uses **Extreme Gradient Boosting (XGBoost)** for binary classification because of its:

- High Accuracy
- Fast Training
- Regularization
- Robustness to Overfitting
- Excellent Performance on Structured Data
